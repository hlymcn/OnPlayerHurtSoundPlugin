# OnPlayerHurtSoundPlugin

## Overview

**OnPlayerHurtSoundPlugin** is an innovative plugin designed for Counter-Strike 2 servers that enhances the gaming experience by playing a custom sound when a player inflicts damage on another player. This plugin is currently in its early beta phase, offering a unique feature that adds an auditory dimension to the game, making it more immersive and engaging for players.

## Key Features

- **Customizable Hurt Sounds:** The plugin allows server administrators to define a list of custom sounds that play when a player hurts another player, adding variety and excitement to each match.
- **Random Sound Playback:** Each time a player hurts another, the plugin randomly selects a sound from the configured list, ensuring variety and unpredictability.
- **Localization Support:** The plugin supports localization, allowing for easy translation into different languages, enhancing the global accessibility of the plugin.

## Usage

To utilize the OnPlayerHurtSoundPlugin, follow these simple steps:

1. **Installation:** Download the plugin from the [GitHub repository](https://github.com/yourusername/OnPlayerHurtSoundPlugin) and place it in your server's plugin directory.
2. **Configuration:** Set up the plugin by configuring the `HurtSounds` list in the plugin's configuration file with the paths to your custom sound files.
3. **Activation:** Once installed and configured, the plugin will automatically play a random hurt sound for each player hurt event.
